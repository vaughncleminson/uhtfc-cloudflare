Wrangler commands:

Login
npx wranger login

Watch tail of worker logfile
npx wrangler tail uhtfc-cloudflare

List sqllite tables
npx wrangler d1 execute d1-uhtfc-cloudflare --remote --command "SELECT name FROM sqlite_master WHERE type='table';"

Export database
npx wrangler d1 export d1-uhtfc-cloudflare --remote --output=./database-exports/export20260131_1.sql

Import database
npx wrangler d1 execute d1-uhtfc-staging --remote --file=./database-exports/export20260131_1.sql
