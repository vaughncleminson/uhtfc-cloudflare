* How to Logout and in again:
npx wrangler logout
npx wrangler login

* How to view tail of server:
npx wrangler tail uhtfc-cloudflare

* How to Export database from cloudflare D1
npx wrangler d1 export d1-uhtfc-cloudflare --remote --output=./database-exports/export10May2026_1321.sql

* Steps to import db to local:
* Note sqlite3 must be installed
* Stop local dev server before import
Local sqlite db is in .wrangler/state/v3/d1/miniflare-D1DatabaseObject:

rm -f .wrangler/state/v3/d1/miniflare-D1DatabaseObject/*.sqlite
npx wrangler d1 execute d1-uhtfc-cloudflare --local --command "SELECT 1" >/dev/null

db_file=$(ls .wrangler/state/v3/d1/miniflare-D1DatabaseObject/*.sqlite | grep -v 'metadata.sqlite')

{ 
  printf 'PRAGMA foreign_keys=OFF;\n'
  cat database-exports/export10May2026_1321.sql
  printf '\nPRAGMA foreign_keys=ON;\n'
} | sqlite3 "$db_file"

* End Steps

* Verify import:
npx wrangler d1 execute d1-uhtfc-cloudflare --local --command "SELECT COUNT(*) AS admins_count FROM admins;"

* Start local dev server after import

* Nice sqlite browser:
sudo apt-get install sqlitebrowser
